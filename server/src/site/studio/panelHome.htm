Studio

Welcome to the OrientDB Studio application. If this is your first time play with the 'demo' database provided in bundle with the official OrientDB distribution.

To Login use "reader/reader" to read records from the database, and "writer/writer" to read, create, update and delete records. To access Server information log into a database and then click on the "Server" tab (top right). At the prompt of credentials use "root" as user and the automatic generated password created under the OrientDB installation directory, in the file "config/orient-server-config.xml".
